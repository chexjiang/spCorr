# Test Data for spCorr Package

A dataset containing example data to be used in examples and testing of
the functions in the spCorr package.

## Usage

``` r
data(test_data)
```

## Format

A list with the following components:

- count_mat:

  A matrix of raw counts

- count_mat:

  A matrix of normalized counts

- gene_list:

  A list of genes

- gene_pair_list:

  A list of gene pairs

- cov_mat:

  A matrix of covariates
